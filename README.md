## üìÅ Dataset Description
The dataset contains Indian housing data specifically for Delhi properties with the following features:

### Features:
| Column | Description |
|--------|-------------|
| `house_type` | Type of property (Apartment, Independent House, Villa, etc.) |
| `house_size` | Size in square feet |
| `location` | Specific locality in Delhi |
| `city` | City (all entries are Delhi) |
| `latitude` | Geographical latitude |
| `longitude` | Geographical longitude |
| `price` | Rental price in INR |
| `currency` | Currency type (all INR) |
| `numBathrooms` | Number of bathrooms |
| `numBalconies` | Number of balconies |
| `isNegotiable` | Whether price is negotiable |
| `priceSqFt` | Price per square foot |
| `verificationDate` | When the listing was posted |
| `description` | Text description of the property |
| `SecurityDeposit` | Security deposit amount |
| `Status` | **Target variable** (Furnished, Semi-Furnished, Unfurnished) |
